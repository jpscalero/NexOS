# NexOS Roadmap - El Futuro de la Distribución

Este documento detalla la visión estratégica y los hitos planificados para NexOS, transformando la base Debian en una plataforma de seguridad de nivel profesional.

## 🧭 Visión Estratégica
NexOS aspira a ser la distribución de referencia para entornos de seguridad "Red Team", combinando una estética premium con una base endurecida (hardened) y automatización avanzada.

---

## 🚀 Hitos v1.1 - Consolidación Pre-Alpha
- [x] **Rebranding Total**: Eliminación de trazas visuales de Debian 13.
- [ ] **Soporte ARM (Raspberry Pi 4/5)**: Generación de imágenes para arquitectura ARM64.
- [ ] **Instalador Gráfico Calamares**: Despliegue simplificado en hardware real.

## 🛡️ Hitos v1.2 - Fase SHIELD (Hardening)
- [ ] **NexOS Shield**: Script de auditoría y remediación basado en estándares CIS.
- [ ] **AppArmor Profiles**: Perfiles de seguridad específicos para herramientas de red.
- [ ] **Kernel Hardening**: Ajustes de `sysctl` para reducir la superficie de ataque.

## 🖥️ Hitos v1.5 - Fase VANGUARD (User Experience)
- [ ] **NexOS Welcome App**: Asistente gráfico de configuración inicial (Qt/Python).
- [ ] **Service Manager Dashboard**: Control centralizado de servicios (SSH, Metasploit, DB).
- [ ] **Optimización de Recursos**: Perfiles de energía y rendimiento dinámicos.

## 🌉 Hitos v1.8 - Fase DEPLOY (Ecosistema)
- [ ] **Repositorio Oficial NexOS**: Distribución de actualizaciones firmadas (`.deb`).
- [ ] **Cloud Ready**: Imágenes certificadas para AWS, Azure y Google Cloud.
- [ ] **Infraestructura como Código (IaC)**: Módulos de Terraform/Ansible para laboratorios.

## 🌌 Visión 2.0 - NEXUS (Certificación)
- [ ] **Nexus-OS Core**: Transición a un núcleo de sistema inmutable (opcional).
- [ ] **Programa de Certificación NexOS**: Ecosistema educativo y profesional.
