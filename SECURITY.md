# Política de Seguridad de NexOS

## Versiones Soportadas

Actualmente, solo proporcionamos parches de seguridad para la última versión lanzada (Release).

| Versión | Soportada |
| ------- | --------- |
| v1.x    | ✅ Soportada |
| < v1.0  | ❌ No Soportada |

## Reportar una Vulnerabilidad

Nos tomamos muy en serio la seguridad de NexOS. Si encuentras una vulnerabilidad, por favor sigue estos pasos:

1. **NO** abras un Issue público en GitHub.
2. Envía un correo electrónico detallado a **jpscalero@protonmail.com**.
3. Incluye una descripción detallada, pasos para reproducir y el impacto potencial.

Responderemos a tu reporte lo antes posible y trabajaremos juntos para solucionar el problema antes de hacerlo público. ¡Gracias por ayudarnos a mantener NexOS seguro!
